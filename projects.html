City 2 City

Tech Used

  • HTML/CSS/JAVASCRIPT/jQUERY/AJAX
  • Bulma-CSS Framework, Teleport-City Data API
  • Open Weather - Weather API

    • This project is designed to compare data between two cities. Data like relative housing costs, quality of life, commutes and more. The user would use this comparison tool to decide on where they would eventually decide to live or relocate. API calls are used for live weather updates and the data will then appear in the form of a butterfly chart.
